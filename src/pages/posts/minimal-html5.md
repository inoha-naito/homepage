# 【HTML】HTML5の最小構成
(実用上での)HTML5文書の最小構成です。  
CSSやJavaScriptの使用は前提としていません。

```
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>文書のタイトル</title>
</head>
<body>
<!-- 文書の本体 -->
</body>
</html>
```
